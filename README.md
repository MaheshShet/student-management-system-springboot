# Student Management System (Spring Boot + Thymeleaf + MySQL)

A real-world **Spring Boot full stack project** to enroll students into courses and their details with a clean layered architecture, form validation, pagination, and secure login.

This repository is **public for learning**, but only the owner maintains the code and merges changes.

This project is created as part of the "Coding with Mahesh" YouTube series.
It is intended for educational and learning purposes.

## Tech Stack

- **Java** (JDK 17/21)
- **Spring Boot**
- **Spring MVC**
- **Thymeleaf**
- **Spring Data JPA (Hibernate)**
- **Spring Security**
- **MySQL**
- **Maven/Gradle** (as per project)

---

## Features

- ✅ Student CRUD (Create, Read, Update, Delete)
- ✅ Server-side form validation
- ✅ Pagination + Sorting
- ✅ Login & Authorization (Spring Security)
- ✅ Global Exception Handling
- ✅ DTO + Entity separation (clean architecture)
- ✅ Responsive UI with Thymeleaf templates
- ✅ Dashboard
